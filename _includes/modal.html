CRM
Mobile Web Application for Customer Relationship Management.

Application was built using Sencha Touch framework and was designed to imitate native iOS Application.

It works on iPhone and iPad using different layout for each platform.

Application works in offline mode (data is stored in local WebSQL database) nad synchronizes on demand with MS SQL Server relational database.

Debt Collector
Android 4.x application for Debt Collection services.

Mobile front end for web server application - main use case is planning tasks for debt collecting firm or department

Application was built using Java native environment for Android with synchronization between client SQLite database and server MS SQL Server.

Sequis
Web Application for equipment maintenance - renting, servicing, inventory.

Application is built using AngularJS framework and Semantic UI library.

CouchDB server and local PouchDB JavaScript library allows offline work and data synchronization between client and server.

All application logic is contained on client side in JavaScript code.

Net Graph
Web Application for network visualization.

Application is built using Sencha Ext JS framework and D3 graphing library

Application uses offline Local Storage for caching network data and JSON REST services for data exchange with web server.

Sequis
Web Application Widget for One2Tribe Gamification Platform

Application is built using AngularJS framework and Bootstrap CSS library.

Java server and REST services are backend for SPA

Application is hosted inside web page that serves as a source for gamification data (ranking, players, missions etc.)
