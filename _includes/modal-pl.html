Net Graph
Aplikacja do wizualizacji sieci.

Aplikacja jest zbudowana przy użyciu Sencha Ext JS i graficznej bibliotekiD3

Aplikacja korzysta z magazynu lokalnego przeglądarki w trybie offline dla buforowania danych i usług sieciowych REST JSON dla wymiany danych z serwerem WWW.

Sequis
Aplikacja do inwentaryzacji sprzętu - wynajem, serwis, magazyn.

Aplikacja jest zbudowana w oparciu AngularJS i Semantic UI.

Serwer CouchDB i lokalna biblioteka PouchDB pozwala na pracę w trybie offline i synchronizację danych między klientem a serwerem.

Cała logika aplikacji zawarta jest po stronie klienta w kodzie JavaScript.

Kolektor długu
Aplikacja Android 4.x do obsługi Windykacji.

Mobilny front-end dla aplikacji internetowej - główne zastosowanie to planowanie zadań dla firm lub działów windykacyjnych.

Aplikacja została zbudowana przy użyciu natywnego środowiska Java dla systemu Android z synchronizacją między kliencką bazą danych SQLite i serwerem MS SQL Server.

CRM
Mobilna aplikacja internetowa do zarządzania relacjami z klientami.

Aplikacja została zbudowana w oparciu oSencha Touch

Aplikacja działa na iPhone i iPad prezentując układ interfejsu dostosowujący się do właściwości każdej platformy.

Aplikacja działa w trybie offline (dane są przechowywane w lokalnej bazie danych WebSQL) na żądanie wykonywana jest synchronizacja z relacyjną bazą danych MS SQL Server.
